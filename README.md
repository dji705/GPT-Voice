# GPT-Voice
# 📞 מערכת מענה AI טלפונית (ימות המשיח + Gemini + Google Sheets)

מערכת המאפשרת למשתמשים לשאול שאלות בטלפון דרך "ימות המשיח", לקבל תשובה מבינה מלאכותית (Gemini), ולתעד את הכל בגיליון גוגל בזמן אמת.

## 📂 מבנה התיקייה

* **`index.js`**: הלב של המערכת. שרת Node.js שמנהל את התקשורת בין ימות המשיח, ה-AI של גוגל וגיליון הנתונים.
* **`package.json`**: רשימת הספריות שהפרויקט צריך (Express, Google-Spreadsheet, וכו').
* **`vercel.json`**: הגדרות הניתוב עבור Vercel, שמאפשרות לשרת לרוץ כ-Serverless Function.
* **`.env` (לא נמצא ב-GitHub)**: קובץ המכיל את המפתחות הסודיים (מוגדר ב-Vercel תחת Environment Variables).

---

## ⚙️ הגדרות נדרשות (Environment Variables)

כדי שהמערכת תעבוד, יש להגדיר ב-Vercel את המשתנים הבאים:

1.  **`GEMINI_API_KEY`**: המפתח שקיבלת מ-Google AI Studio.
2.  **`SHEET_ID`**: המזהה הייחודי של גיליון הגוגל שלך (מתוך הכתובת של הדפדפן).
3.  **`GOOGLE_SERVICE_ACCOUNT`**: תוכן קובץ ה-JSON של חשבון השירות (Service Account) שמאפשר גישה לגיליון.

## 🚀 איך זה עובד?

### שלב 1: שליחת שאלה (`/api/ymotAskAI`)
* ימות המשיח פונים לכתובת זו עם מספר הטלפון והטקסט שהמשתמש הקיש.
* השרת שולח את השאלה ל-**Gemini AI**.
* התשובה מתקבלת, מנוקה מתווים מיוחדים ונרשמת בשורה חדשה בגיליון גוגל.

### שלב 2: קריאת תשובה (`/api/ymotReadAI`)
* ימות המשיח פונים לכתובת זו.
* השרת מחפש בגיליון את השורה האחרונה שמתאימה למספר הטלפון של המחייג.
* אם נמצאה תשובה, היא מוחזרת כטקסט להקראה (TTS).
